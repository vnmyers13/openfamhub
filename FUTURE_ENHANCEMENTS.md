# HomeHub — Future Enhancements

This file tracks feature requests, enhancement ideas, and post-MVP capabilities. It is updated before every release by running Phase 1 of the release checklist, which fetches open GitHub issues labelled `enhancement` or `feature request` automatically.

Items are sourced from:
- GitHub Issues labelled `enhancement` or `feature request`
- The post-MVP backlog from the product requirements document
- Community suggestions

## High Priority (Next Phase)

- [ ] Task and chore management module — assign recurring tasks to family members, Kanban board, photo evidence on completion
- [ ] Custom reusable lists — packing lists, pre-sleepover checklists, grocery lists with shopping mode
- [ ] Photo albums and wall slideshow — replace WallPhotoPlaceholder with cycling photo display
- [ ] Push notifications (VAPID) — overdue task alerts, event reminders to PWA on phones

---

## Medium Priority

- [ ] Routines builder and runner — group tasks into morning/bedtime/after-school routines with step-by-step guided mode
- [ ] Meal planning and recipe library — weekly meal planner, recipe library with dietary tags, auto-generate shopping list
- [ ] Family announcements board — fridge-replacement sticky notes with emoji reactions
- [ ] Weather widget — Open-Meteo (no API key), current + 5-day forecast on wall and dashboard
- [ ] Reward and points system — assign point values to tasks, accumulate points, define reward tiers for children

## Lower Priority / Community Requests

- [ ] Wall board panel editor — admin drag-and-drop layout editor for the wall display

- [ ] Homework tracker — subject/assignment tracking for school-age children integrated with calendar
- [ ] Family wiki / reference page — emergency contacts, insurance, school schedules, WiFi passwords
- [ ] Allowance tracker — monetary allowance tied to chore completion, balance and payment ledger
- [ ] Pet care tracker — feeding schedules, vet appointments, medication reminders
- [ ] Vehicle maintenance tracker — service history, upcoming maintenance, registration renewal alerts
- [ ] Home Assistant integration — display sensor values on wall, trigger automations from routines
- [ ] Voice assistant webhooks — Alexa / Google Home voice queries for calendar and tasks
